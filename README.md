# PM Orquestra
Projeto Gestão de Projetos

## Objetivo

Framework para cração de projetos que contempla:

* PI – Procedimentos de Iniciação
    * PM Canvas
    * TAP 
* PP – Procedimentos de Planejamento
    * Escopo
    * EAP
    * Cronograma
    * Gestão de Partes Interessadas
    * 5W2H
    * Plano do Projeto
* PA – Procedimentos de Acompanhamento (Execução/controle)
    * Controle de Execução
* PEF – Procedimento de Encerramento de Fase
    * Gerenciamento de Mudanç
* PEP – Procedimento de Encerramento de Projeto
    * Lições Aprendidas
    * Base de Conhecimento
